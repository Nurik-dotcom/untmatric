## Как запустить:

### 1️⃣ В Godot Editor:

```
1. Откройте Godot 4.5
2. Откройте проект (папка: untmatric/)
3. Дождитесь переиндексации (автоматическое сканирование файлов)
4. Нажмите F5 или Play
5. Нажмите кнопку "LEARN" в главном меню
```

### 2️⃣ Ожидаемый результат:

```
MainMenu (главное меню)
  ↓ [LEARN]
LearnSelect (выбор учебных модулей)
  ├─ [Двоичная система] → TutorialBinary.tscn
  ├─ [Шестнадцатеричная система] → TutorialHexadecimal.tscn
  ├─ [Логические ворота] → TutorialLogicGates.tscn
  └─ [Матричные головоломки] → TutorialMatrix.tscn
```

### 3️⃣ Навигация внутри учебного модуля:

- **← Назад** - предыдущий шаг
- **Далее →** - следующий шаг
- **Пропустить** - завершить модуль и вернуться
- **На главную** - вернуться в LearnSelect
